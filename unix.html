Unix's Command Cheat Sheet

pwd Show the full path to the current directory

ls List files and directories in the current directory

ls -al List all files and directories including hidden files in long format

man command_name Display the manual for command_name

cd folder1 Go to folder1 directory

cd .. Go to the parent directory

cd / Go to the root directory

cd Go to the current directory

mkdir folder1 Make a new directory named folder1

rmdir folder1 Delete folder1 directory

cp file1 file2 Copy file1 and name it as file2

mv file1 file2 Move file1 and name it as file2

rm file1 Delete file1

rm * Delete all files in the current directory

rm proj?.html Delete all html files that have 5 letters and start with proj

chmod 755 folder1 Set permission of folder1 to full access for owner, read and execute for group and everyone else

chmod 644 file1 Set permission of file1 to read and write for owner, read for group and everyone else

chmod 700 folder1 Set permission of folder1 to full access for owner, and block all access from group and everyone else
